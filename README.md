# Tiles Gallery

A modern and responsive tile gallery web application where users can explore premium tiles, search designs, view details, manage profiles, and add tiles to cart.

---

## Live Website

https://your-live-link.vercel.app

---

## Project Purpose

The purpose of this project is to build a modern tile showcase platform using Next.js App Router with authentication, protected routes, responsive UI, and dynamic tile management features.

---

## Design inspired by inverto market templates

## Key Features

- Responsive design for mobile, tablet, and desktop
- User Authentication with Email/Password & Google Login
- Protected Private Routes
- Dynamic Tile Details Page
- Featured Tiles Section
- Tile Search Functionality
- Add To Cart System
- Order History Page
- User Profile Management
- Update User Information
- Beautiful Swiper Slider
- Loading Spinner & Toast Notifications
- Custom 404 Not Found Page
- Environment Variables Protection
